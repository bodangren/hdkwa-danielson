# Spec: Domain 4 Content Hub

## Problem
Domain 4 (Professional Responsibilities) is not implemented. This completes the Danielson Framework coverage.

## Solution
Build the Domain 4 content hub with subdomains, techniques, videos, and self-assessments matching the existing Domain 3 pattern.

## Acceptance Criteria
- [ ] Domain 4 hub page with subdomains: 4a (Reflecting on Teaching), 4b (Maintaining Accurate Records), 4c (Communicating with Families), 4d (Participating in Professional Community), 4e (Growing and Developing Professionally), 4f (Showing Professionalism)
- [ ] At least 2 techniques per subdomain with title, description, video embed, guide
- [ ] Self-assessment before/after for each technique
- [ ] localStorage integration with existing progress schema
- [ ] Navigation links from main menu and home page

## Out of Scope
- Professional development credit tracking
- External certification integration
- Peer review features
