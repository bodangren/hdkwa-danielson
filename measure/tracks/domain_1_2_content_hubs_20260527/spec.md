# Spec: Domain 1 & 2 Content Hubs

## Problem
Only Domain 3 (Instruction) has a content hub. Domains 1 (Planning) and 2 (Classroom Environment) are stubbed or empty. Teachers cannot access techniques for these critical domains.

## Solution
Build content hubs for Domain 1 and Domain 2 with the same structure as Domain 3: subdomain pages, technique detail pages with videos, self-assessments, and reflections.

## Acceptance Criteria
- [ ] Domain 1 hub page with subdomains: 1a (Instructional Outcomes), 1b (Designing Coherent Instruction), 1c (Designing Student Assessments), 1d (Using Resources), 1e (Designing Coherent Instruction)
- [ ] Domain 2 hub page with subdomains: 2a (Creating Environment of Respect), 2b (Establishing Culture for Learning), 2c (Managing Classroom Procedures), 2d (Managing Student Behavior), 2e (Organizing Physical Space)
- [ ] At least 3 techniques per subdomain with: title, description, video embed, step-by-step guide
- [ ] Self-assessment before/after for each technique (1-5 rating + goal/reflection)
- [ ] Data stored in localStorage with the same schema as Domain 3
- [ ] Navigation updated to include Domains 1 & 2

## Out of Scope
- Admin content management UI
- Backend persistence
- User accounts
