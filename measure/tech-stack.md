# Tech Stack

## Core Framework
- **Next.js (App Router):** For robust routing, static generation (SSG), and modern developer experience.
- **TypeScript:** To ensure type safety and maintainability.

## Styling & UI
- **Tailwind CSS:** For rapid, responsive UI development.
- **Lucide React:** For consistent, professional iconography.
- **Shadcn UI (optional/partial):** For high-quality, accessible components if needed.

## State & Storage
- **Local Storage:** Used for tracking teacher progress (completed strategies, badges) without a backend.
- **Zustand (optional):** For simple client-side state management if logic becomes complex.

## Deployment
- **GitHub Pages:** Static export (`next export` / `output: 'export'`) for free, reliable hosting.
- **GitHub Actions:** For automated build and deployment pipelines.

## Multimedia
- **YouTube Embed API:** For seamless video integration from the repo's curated video index.
